gas-sim-pro
Draw a room, place sensors, watch gas diffuse — an ML model trained on 1.8 million simulations pinpoints the invisible leak source in real time.
The Problem

Gas leaks are invisible. Conventional detectors tell you gas is present — never where it came from. Existing solutions require dense sensor grids or expert inspectors on site.

The Engine

FTCS diffusion + wind advection on a 100×100 grid generates training data at scale. XGBoost predicts leak origin from sparse readings. MAE 0.037 cells on test set (simulation-to-simulation). Real-world detection is significantly harder — but the pipeline is proven end-to-end. FastAPI on Cloud Run, <50ms.

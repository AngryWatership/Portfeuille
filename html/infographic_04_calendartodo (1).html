Calendar ToDo
Goals as a live dependency graph — complete one node and the next unlocks, annul one and everything downstream cascades automatically. Every task has a bar on a zoomable timeline with a live "now" indicator so deadlines feel real, not abstract.
The Problem

To-do lists have no memory of why a task exists. Project management tools are too rigid for personal goals. Neither model the logical dependencies between objectives — so when one thing changes, nothing else adjusts.

The Engine

8 cascade rules govern state propagation: Completion, XOR Suspension, Playhead Pin, Goal Termination, Parent Expansion, Upward Fatal Cascade, Soft Completion, Orphan Rescue. 4 node types, 4 link types. XOR groups preserve mutually exclusive paths as strategic memory. 32 unit tests. Vue 3 + Pinia.
